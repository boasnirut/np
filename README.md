# เว็บไซต์โรงเรียนบ้านน้ำพร

เว็บไซต์ประชาสัมพันธ์โรงเรียนบ้านน้ำพร พัฒนาด้วย React + Vite รองรับทั้งคอมพิวเตอร์ แท็บเล็ต และโทรศัพท์มือถือ พร้อมนำขึ้น Vercel

- เว็บไซต์: https://np-five-pi.vercel.app
- GitHub: https://github.com/boasnirut/np

## เริ่มใช้งาน

```bash
npm install
npm run dev
```

เปิด `http://localhost:5173`

## สร้างไฟล์สำหรับเผยแพร่

```bash
npm run build
```

Vercel จะตรวจพบ Vite โดยอัตโนมัติและใช้คำสั่ง `npm run build` ผลลัพธ์จะอยู่ในโฟลเดอร์ `dist`

## แก้ไขข้อมูลเว็บไซต์

- ข่าว กิจกรรม บริการ ที่อยู่ และเบอร์โทร: `src/content.js`
- ข้อความและโครงสร้างหน้าเว็บ: `src/App.jsx`
- สี รูปแบบ และการแสดงผลบนมือถือ: `src/styles.css`
- โลโก้และไอคอนเว็บไซต์: `public/np.png`, `public/favicon.png`

## ข้อมูลที่ควรอัปเดตก่อนประกาศใช้งานจริง

รายการข่าวและกิจกรรมในเว็บไซต์เป็นเนื้อหาเริ่มต้น ควรตรวจสอบวันเวลาและเปลี่ยนเป็นข้อมูลจริงจากโรงเรียน รวมถึงเพิ่มลิงก์ Facebook อีเมล เอกสาร และระบบสารสนเทศในปุ่มที่เกี่ยวข้อง
